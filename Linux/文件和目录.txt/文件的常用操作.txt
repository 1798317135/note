 -- ~ 这个符号表示系统的家目录

-- ls 查看当前文件夹里面的内容 
    -a 查看包含隐藏文件
    -l 显示每个的详细信息 -h 显示大小的单位

-- pwd 查看当前的文件夹

-- cd 【目录路径】 切换目录

-- touch【文件明】 创建文件

-- mkdir[文件夹] 创建文件夹
    -p 递归创建嵌套文件夹

-- rm [文件明] 删除文件明 
    -r 参数可以删除文件夹
    -f 强制删除

-- tree 树状的显示文件结构
    - ~ 显示根目录
    -d 不看文件 只看目录

-- cp 复制文件
    -i 覆盖之前的提示(可以用于多个命令的覆盖提示)
    -r 可以复制目录

-- mv 把源文件 移动到目标文件 (也可以用于重命名操作)
    -i 可以覆盖提示

-- cat 查看文件内容（一次性加载全部内容）
    -b 显示行号，有内容的
    -n 包括空行的行号也显示

-- more 查看(每次查看一部分)
    
-- grep 搜索工具
    -i 忽略大小写
    -v 取反搜索
    -n 显示行号

-- echo 输出文件 和 重定向到文件
    > 覆盖文件的内容
    >> 追加到文件后面

-- 重定向 
    > 覆盖文件的内容
    >> 追加到文件后面
    | 管道命令操作符
    可以将两个命令分开
    左边的输入作为右边的输入
    连续依次类推

-- tailf 阻塞查看文件内容

-- find 路径   查找文件
    -name "" 指定条件

-- sed -i "/正则/"d 文件          他可以删除文件指定的内容


-- clear 清屏
