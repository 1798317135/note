-- chmod 修改用户和组 对文件和目录的权限
    chmod +/- rwx 目录|文件  增加或者减少文件的权限
    -R 数字权限 文件|目录
        755 
        644
        777 
        第一个数组 拥有着权限
        第二个 组权限
        第三个 其他用户权限

        
-- chgrp 修改组
    -R 组名 文件名|目录

-- chown 用户名 文件|目录   修改拥有着

-- ls -l 可查看文件和目录的权限
    -d 目录
    - 文件
    后面的权限 三个一组 
    前三个 是 拥有者的权限
    中间三个 是 组用户的权限
    后面三个是 其他用户的权限
    然后那一排数字
    硬链接数 （可以访问这个文件的路径数）
    后面是 拥有着
     然后是 组 
     然后 是 大小日期

-- root 是超级管理员
    用于系统的维护和管理
    sudo 表示用root用户身份

-- 通常使用标准用户
    groupadd 组名 添加组

    /etc/group 组用户 储存在这里

    cat /etc/group 可以查询组信息

    groupdel 组名 删除组

    chgrp -R 组名 文件/目录，名 他会递归的修改文件夹内部的所属组

-- useradd 新建用户
    -m 自动建立用户的家目录
    -g 指定用户的组否则会建立一个同名的组

-- passwd 用户名 设置密码

    /etc/passwd/ 用户储存在这里

    /usr/bin/passwd 这是可以修改用户密码的程序

    /home 用户的家目录 默认是在这里

-- userdel 用户名 删除用户
    -r 自动删除家目录

-- id 用户名   查看用户的信息
    uid 用户代号
    gid 组代号
    x 有密码加密

-- who 查看连接主机的客户端信息

-- whoami 现在登陆的用户

-- usermod 可以给创建的用户 添加附加组 给予更多的权限
    -g 修改的组
    -G 增加用户的附加组s
    -s /bin/bash 用户名 #更改用户shell

-- which 程序  他可以查看程序的路径

-- su 切换用户
    - 可以切换用户的家目录

-- exit 返回到用户
